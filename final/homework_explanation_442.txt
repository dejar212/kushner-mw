Пояснение к коду (Зачет, Задача 2.7 Сороколетов)

Уравнение: u_t + u_x + u + a = 0.

Задача: найти конечномерную динамику первого порядка и построить точное решение, отвечающее этой динамике. Метод тот же, что разбирался на лекциях и применялся в работе Оруджевой: ищем инвариантное относительно t-потока ОДУ первого порядка по x, u_x = f(x, u), затем редуцируем эволюционное уравнение на это инвариантное подмногообразие в J^1.

restart;
with(DifferentialGeometry):with(JetCalculus):with(Tools):with(PDETools):
- Подключаем стандартные пакеты дифференциальной геометрии, исчисления джетов, инструментов конвертации и пакет уравнений с частными производными.

Preferences("JetNotation", "JetNotation2");
- Включаем нотацию y[0], y[1], y[2], ... для функции и её производных в джет-пространстве.

DGsetup([x], [y], M, 1, verbose);
- Задаем многообразие струй первого порядка J^1 с независимой переменной x и зависимой y. Координаты: x, y[0], y[1].

E:=diff(u(t,x),t)+diff(u(t,x),x)+u(t,x)+a;
- Левая часть исходного уравнения.

phi:=-y[1]-y[0]-a;
- Эволюционная часть: u_t = phi(x, u, u_x). Здесь y[0] это u, y[1] это u_x. Получаем phi = -y[1] - y[0] - a.

ode:=diff(y(x),x)-f(x,y(x));
F:=convert(ode,DGjet);
- Записываем кандидата на инвариантное ОДУ первого порядка u_x = f(x, u) в форме y[1] - f(x, y[0]) = 0 и переводим в джет-нотацию.

omega:=evalDG(dy[0]-f(x, y[0])*dx);
- Контактная (структурная) 1-форма, аннулирующая распределение, заданное ОДУ y[1] = f. На решениях ОДУ omega = 0.

S:=evalDG(phi*D_y[0]);
- Эволюционное векторное поле, отвечающее уравнению. Его компонента вдоль ∂/∂y[0] есть phi.

sub_y1:={y[1]=f(x, y[0])};
S1:=eval(S,sub_y1);
- Ограничение поля S на подмногообразие y[1] = f.

evalDG(LieDerivative(S1,omega) &w omega);
- Условие инвариантности (совместимости): инвариантное подмногообразие y[1] = f сохраняется t-потоком тогда и только тогда, когда L_S(omega) ∧ omega = 0 на этом подмногообразии. Это даёт уравнение на f.

G:=-diff(f(x,y[0]),x)+diff(f(x,y[0]),y[0])*(y[0]+a)-f(x,y[0]);
- Раскрытие условия совместимости даёт линейное уравнение в частных производных первого порядка на f(x, y[0]):
  -f_x + f_{y[0]} (y[0] + a) - f = 0,
  то есть f_x + f = f_{y[0]} (y[0] + a).

pdsolve(G);
- Решаем это PDE методом характеристик. Характеристики dx/1 = dy[0]/(-(y[0]+a)) = df/(-f) дают первые интегралы (y[0]+a) e^x и f e^x. Общий вид: f(x, y[0]) = e^{-x} * Phi((y[0]+a) e^x), где Phi произвольная функция.

sub_f:={f(x, y[0]) = y[0]+a};
- Выбираем простейшую нетривиальную динамику: Phi(s) = s, то есть f = y[0] + a. Соответствующее инвариантное ОДУ: u_x = u + a.

F1:=eval(F,sub_f);
dsolve(convert(F1,DGdiff));
- Решаем выбранное ОДУ по x. Общее решение по x: y[0] = C * exp(x) - a, где C произвольная константа (которая в полном решении будет зависеть от t).

S2:=eval(-S1,sub_f);
- Поле эволюции на одномерной инвариантной поверхности после подстановки f. Знак минус согласует ориентацию потока с t-параметром.

Xi:=simplify(Flow(S2,t));
- Поток векторного поля S2 на параметр t. Он описывает, как координата y[0] меняется со временем на инвариантном многообразии.

v:=[y[0]-C*exp(x)+a];
solve(Pullback(Xi,v),y[0]);
- Берем начальный профиль y[0]|_{t=0} = C exp(x) - a из решения x-овского ОДУ. Откатив по потоку Xi, выражаем y[0](t, x). Результат: y[0] = C exp(x - 2t) - a.

sub_u:={u(t,x)=C*exp(x-2*t)-a};
simplify(eval(E,sub_u));
- Записываем найденное точное решение и подставляем в исходное уравнение для проверки. Результат должен быть тождественно 0.

Проверка вручную: u = C e^{x-2t} - a, u_t = -2 C e^{x-2t}, u_x = C e^{x-2t}, u + a = C e^{x-2t}.
Сумма: -2 C e^{x-2t} + C e^{x-2t} + C e^{x-2t} = 0. Уравнение выполнено.

plot([seq(...)], ...);
plot3d(...);
- Визуализация полученного семейства решений при a = 1, C = 1 для t = 0, 1, 2, 3, 4 и трёхмерный график u(t, x).

Вывод:
- Уравнение совместимости для динамики первого порядка u_x = f(x, u): f_x + f = f_{y[0]} (y[0] + a). Его общее решение f = e^{-x} Phi((u+a) e^x).
- Простейший выбор f = u + a даёт инвариантное ОДУ u_x = u + a, общее решение которого по x есть u = C exp(x) - a.
- Редукция эволюции на параметр C даёт C(t) = C_0 exp(-2 t), что приводит к точному решению исходного уравнения
  u(t, x) = C exp(x - 2 t) - a,
  отвечающему найденной одномерной конечномерной динамике.
