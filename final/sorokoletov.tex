\begin{center}
{\large Задача 2.7 (Сороколетов)}\\[4pt]
$u_t + u_x + u + a = 0$
\end{center}

\section*{Постановка}

Уравнение эволюционное: $u_t = -u_x - u - a$. Конечномерная динамика порядка $n$ задаётся ОДУ по $x$ вида
\[
u^{(n)} = f(x, u, u_x, \dots, u^{(n-1)}),
\]
которое инвариантно относительно $t$-потока уравнения. На таком инвариантном подмногообразии общее решение $x$-овского ОДУ зависит от $n$ постоянных $c_1, \dots, c_n$. Подставив $u = U(x; c_1(t), \dots, c_n(t))$ в исходное уравнение, получим систему ОДУ на $c_i(t)$ — это и есть редуцированная динамика.

\section*{Динамика порядка 1: $u_x = f(x, u)$}

Условие инвариантности. Дифференцируем связь $u_x - f(x,u) = 0$ по $t$ и подставляем уравнение:
\[
u_{tx} = f_u \, u_t = f_u(-u_x - u - a) = f_u(-f - u - a).
\]
С другой стороны, $u_{tx} = D_x u_t = -u_{xx} - u_x$. На связи $u_x = f$ имеем $u_{xx} = D_x f = f_x + f_u u_x = f_x + f_u f$. Поэтому
\[
u_{tx} = -(f_x + f_u f) - f.
\]
Приравнивая два выражения для $u_{tx}$, получаем уравнение совместимости
\[
-f_x - f_u f - f = -f_u f - f_u(u + a),
\]
откуда
\[
\boxed{\; f_x + f = f_u\,(u + a). \;}
\]

\subsection*{Решение уравнения совместимости}

Линейное уравнение в частных производных первого порядка. Характеристики:
\[
\frac{dx}{1} = \frac{du}{-(u+a)} = \frac{df}{-f}.
\]
Из первой пары: $(u+a)\, e^{x} = C_1$. Из второй: $f\, e^{x} = C_2$. Значит общее решение
\[
f(x,u) = e^{-x}\, \varphi\!\left((u+a)\, e^{x}\right),
\]
где $\varphi$ произвольная функция одного переменного. Каждая такая функция $\varphi$ задаёт инвариантное ОДУ
\[
u_x = e^{-x}\, \varphi\!\left((u+a)\, e^{x}\right),
\]
то есть конечномерную динамику порядка 1.

\subsection*{Простейшие случаи}

Случай A: $\varphi \equiv 0$, то есть $u_x = 0$. Тогда $u = c(t)$, и подстановка в уравнение даёт
\[
c'(t) + c(t) + a = 0 \quad\Longrightarrow\quad c(t) = (c_0 + a)\, e^{-t} - a.
\]
Точное решение: $u(x, t) = (c_0 + a)\, e^{-t} - a$.

Случай B: $\varphi(s) = s$, то есть $u_x = u + a$. Общее решение по $x$: $u = c(t)\, e^{x} - a$. Подставляем:
\[
u_t = c'(t)\, e^{x}, \quad u_x = c(t)\, e^{x}, \quad u + a = c(t)\, e^{x},
\]
\[
c'\, e^{x} + c\, e^{x} + c\, e^{x} = 0 \quad\Longrightarrow\quad c'(t) = -2 c(t),
\]
откуда $c(t) = c_0\, e^{-2 t}$ и точное решение
\[
u(x,t) = c_0\, e^{x - 2 t} - a.
\]

Случай C: $\varphi \equiv \alpha$ (постоянная), то есть $u_x = \alpha\, e^{-x}$. Интеграл по $x$: $u = -\alpha\, e^{-x} + c(t)$. Подставляем:
\[
u_t = c'(t), \quad u_x = \alpha\, e^{-x},
\]
\[
c'(t) + \alpha\, e^{-x} + (-\alpha\, e^{-x} + c(t)) + a = 0
\quad\Longrightarrow\quad c'(t) + c(t) + a = 0,
\]
откуда $c(t) = (c_0 + a)\, e^{-t} - a$ и
\[
u(x,t) = -\alpha\, e^{-x} + (c_0 + a)\, e^{-t} - a.
\]

\section*{Динамика порядка 2: $u_{xx} = 0$}

Проверяем инвариантность. Дифференцируем $u_{xx} = 0$ дважды по $x$ от уравнения и подставляем. Из $u_t = -u_x - u - a$ имеем $u_{txx} = -u_{xxx} - u_{xx} = 0$ на связи $u_{xx} = 0$ (так как $u_{xxx} = D_x u_{xx} = 0$). Связь сохраняется.

Общее решение ОДУ $u_{xx} = 0$: $u = c_1(t) + c_2(t)\, x$. Подставляем:
\[
c_1'(t) + c_2'(t)\, x + c_2(t) + c_1(t) + c_2(t)\, x + a = 0.
\]
Приравнивая коэффициенты при степенях $x$:
\[
\begin{cases}
c_2'(t) + c_2(t) = 0,\\
c_1'(t) + c_1(t) + c_2(t) + a = 0.
\end{cases}
\]
Решение: $c_2(t) = A\, e^{-t}$, далее
\[
c_1'(t) + c_1(t) = -A\, e^{-t} - a \quad\Longrightarrow\quad c_1(t) = (B - A t)\, e^{-t} - a.
\]
Точное решение, отвечающее этой 2-мерной динамике:
\[
u(x, t) = A\, e^{-t}\, x + (B - A t)\, e^{-t} - a.
\]

\section*{Сводка}

1. Уравнение совместимости для динамики порядка 1: $f_x + f = f_u(u + a)$, общее решение $f = e^{-x}\, \varphi((u+a)\, e^{x})$.

2. Простейшие 1-параметрические семейства решений: $u = C e^{-t} - a$, $u = C\, e^{x - 2 t} - a$, $u = -\alpha\, e^{-x} + (C + a)\, e^{-t} - a$.

3. Полиномиальная по $x$ динамика порядка 2 даёт двухпараметрическое семейство $u = A\, e^{-t}\, x + (B - A t)\, e^{-t} - a$.
