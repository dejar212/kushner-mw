# kushner-mw

Домашние и зачетные работы по курсу Кушнера (геометрия дифференциальных уравнений, управляемость, конечномерные динамики). Репозиторий рассчитан на использование разными студентами: каждый кладет свои файлы в собственную папку и подставляет свою фамилию и группу в имена файлов согласно шаблону из `AGENT_INSTRUCTIONS.md`.

## Структура

```
.
├── AGENT_INSTRUCTIONS.md           строгая инструкция для агента
├── rules-of-work.txt               правила оформления текста
├── templates/                      шаблоны .mw, .txt и пояснения
├── examples/                       принятые преподавателем работы (эталон стиля)
├── hw1/                            домашка 1: машина Дубинса
├── hw2/                            домашка 2: управляемость на сфере S^2
└── final/                          зачет: конечномерная динамика для u_t + u_x + u + a = 0
```

В каждой папке задания лежат три файла одного формата:

- `homework_maple_<surname_latin>_<group>.mw` рабочий лист Maple
- `homework_maple_<surname_latin>_<group>.txt` тот же код в виде plain text
- `homework_explanation_<group>.txt` построчное пояснение

`<surname_latin>` и `<group>` агент должен уточнить у пользователя в самом начале работы.

## Как агенту решать новое задание

1. Прочитать `AGENT_INSTRUCTIONS.md` целиком.
2. Прочитать `rules-of-work.txt` (правила оформления).
3. Посмотреть `examples/` чтобы понять принятый преподавателем стиль.
4. Скопировать соответствующий шаблон из `templates/` и адаптировать под условие задачи.
5. Сложить результат в новую папку (`hw3/`, `final/`, и т. д.) тремя файлами в том же формате.

## Принятые работы

- hw1 машина Дубинса, оформлено как композиция потоков X, Y, [X, Y].
- hw2 управляемость на сфере S^2, оформлено по теореме Рашевского Чоу.
- final задача 2.7 (Сороколетов), $u_t + u_x + u + a = 0$, решена через инвариантное ОДУ $u_x = f(x, u)$ и условие совместимости $L_S(\omega) \wedge \omega = 0$. Точное решение $u(t, x) = C e^{x - 2 t} - a$.
